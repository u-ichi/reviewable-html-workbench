# Reviewable HTML Workbench

Claude Code / Codex CLI の両方から使うHTMLレビュー用plugin。

このrepoは2つのskillを含む。

- `visual-html-renderer`: 図示つきHTML生成、Preview Runtime、コメントUI
- `reviewable-design-doc`: レビュー可能な設計資料、コメント取り込み、agent返信、設計反映

詳細な開発フローは `docs/development-flow.md` を参照。

設計ドキュメントは `docs/design.html` に配置する。
